Introduction

This page demonstrate the use of ie_expand_select_width to fix select box width issue on IE8. For demo purpose the script is active for all browsers, but in production it should only be active for IE8. Because its behavior will be slightly different (poorer) on other browsers.

Contrary to several other solutions, this script keep the page layout intact.

Usage

Include the script only for IE: 
<script type="text/javascript" src="//ajax.googleapis.com/ajax/libs/jquery/1.8/jquery.min.js"></script>
<script type="text/javascript" src="//ajax.googleapis.com/ajax/libs/jqueryui/1.8.22/jquery-ui.min.js"></script>
<!--[if lt IE 9]>
	<script type="text/javascript" src="ie_expand_select_width.js"></script>
	<script type="text/javascript">
		$(document).ready(function() {
			$('select').ieExpandSelectWidth();
		});
	</script>
<![endif]-->
